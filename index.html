Env - Secure File Sharing

A secure file sharing application built with Next.js, React, and TypeScript. It allows users to upload, encrypt, and share files via email or a secure link. The backend leverages Prisma ORM with a PostgreSQL database, and integrates with AWS services like S3 for file storage, SES for sending emails, and KMS for cryptographic key management. Authentication is handled using NextAuth.js, providing a robust and secure user experience.
